| References | Methods         | What's it for?                                                      |
|------------|-----------------|---------------------------------------------------------------------|
| React      | createRoot()    | create a root to display React components inside a browser DOM node |
| ReactDOM   | createElement() | create a virtual DOM                                                |
| ReactDOM   | render()        | render React component                                              |
